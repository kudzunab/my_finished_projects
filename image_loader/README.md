# Загрузчик изображений

Проект представляет собой загрузчик изображений по их прямым ссылкам. Работает в условиях неустойчивых сетей с возможностью использования proxy.

## 🛠️ Технологии
* **Язык:** Python 3
* **Расширения:** asyncio

## 📦 Сборка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите скрипт:
   ```bash
   python image_loader.py
   ```
