# Бэкенд сетевой версии игры

Проект представляет собой серверное приложение на Flask, демонстрирующее функционал для параллельного ведения изолированных игровых сессий с компьютером для множества пользователей.

## 🛠️ Технологии
* **Язык:** Python 3
* **Фреймворк:** Flask

## 📦 Сборка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите нужные библиотеки.
3. Запустите сервер:
   ```bash
   python main.py
   ```
.
