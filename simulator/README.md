# Симулятор

Проект представляет собой симулятор процесса, координирующего действия нескольких процессов, которые отрабатывают определенные паттерны и по завершении выводят статистику результата своей работы.

## 🛠️ Технологии
* **Язык:** Python 3
* **Расширения:** multiprocessing

## 📦 Сборка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите скрипт:
   ```bash
   python simulator.py
   ```
