# Бэкенд сетевой версии игры

Проект представляет собой серверное приложение на Flask, использующее Basic авторизацию, а также функционал для хранения истории сессий и формирования таблицы лидеров на основе SQL-запросов. Ключевые возможности включают управление парой токенов, защиту эндпоинтов и расчет рейтингов пользователей.

## 🛠️ Технологии
* **Язык:** Python 3
* **Фреймворк:** Flask
* **Расширения:** SQLAlchemy

## 📦 Сборка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите сервер:
   ```bash
   python main.py
   ```
   В случае отсутствия базы данных PostgreSQL, параметры управления которой указаны в корне проекта с помощью файла `.env` (более детальная информация представлена в файле [read_me](./src/read_me.txt)), проект запустится через SQLite.
