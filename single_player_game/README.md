# Игра-бродилка

Проект представляет собой игру-бродилку на фиксированное число уровней с последующим усложнением процесса игры (появление более мощных монстров, уменьшение лута).

## 🛠️ Технологии
* **Язык:** Python 3

## 📦 Сборка и запуск

1. Склонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите скрипт:
   ```bash
   python main.py
   ```
